## 🌟 Project Overview

Tired of staring at a blank `README.md` file? Say hello to **README.exe**! This innovative tool leverages the power of Google's Gemini API to automate the creation of both your personal GitHub Profile README and comprehensive Project READMEs. With a user-friendly interface and a plethora of customization options, you can now craft professional and eye-catching READMEs without breaking a sweat. 😎

## ✨ Key Features

Get ready to be amazed by what README.exe can do! 🤩

*   **Profile README Generator:**
    *   A slick 7-step form to guide you through creating your perfect Profile README:
        1.  **Identity:** Your name, GitHub username, and bio.
        2.  **About:** A more detailed description of yourself.
        3.  **Skills:** Showcase your expertise with a vast library of 50+ skill badges! 🧠
        4.  **Stats:** Integrate GitHub Stats cards, streak, trophies, and top languages. 🏆
        5.  **Social:** Add links to your social media profiles with stylish badges. 🌐
        6.  **Extras:** Include cool widgets like Typing SVG banners and profile views counters. 📈
        7.  **Style:** Customize the look with 3 header styles (Simple, Banner, Animated) and 10 stats themes (Radical, Tokyo Night, Dracula, Nord, and more!). 🎨
*   **AI-Powered Project README Generator:**
    *   Simply input your project details and let the Google Gemini API work its magic to instantly generate a comprehensive Project README. 🤖
*   **Live Preview:**
    *   See your README come to life in real-time with a live markdown preview.
    *   Toggle to view the raw markdown code. 👀
*   **Effortless Export:**
    *   One-click copy to clipboard or download your README.md file. 📥
*   **Robust AI Integration:**
    *   Gemini API keys are securely hidden server-side using Vercel Serverless Functions. 🔒
    *   Intelligent model fallback: automatically tries `gemini-2.5-flash`, `lite`, and `pro` in order for optimal performance. 🔄
*   **Sleek UI/UX:**
    *   Custom animated cursor and a dot-grid dark UI theme for a modern and engaging experience. 🌌

## 📖 How to Use

It's super simple to get started! 🎉

1.  **Visit the Site:** Head over to [https://git-readme-generator-jr5q.vercel.app/](https://git-readme-generator-jr5q.vercel.app/)
2.  **Choose Your Path:** Select either "Profile README" or "Project README" from the landing screen.
    *   **For Profile README:**
        *   Navigate through the 7 steps on the left panel, filling in your details.
        *   Watch your README update live on the right panel.
        *   Once you're happy, simply click "Copy" or "Download" to get your `README.md` file.
    *   **For Project README:**
        *   Fill in the necessary project details.
        *   Click "Generate with AI".
        *   Let Gemini work its magic and instantly receive a professionally written README!

## 🚀 Installation (For Developers)

Wanna run this awesomeness locally? Here’s how!

1.  **Clone the repository:**
    ```bash
    git clone https://github.com/instai4/GIT-README-GENERATOR.git
    cd GIT-README-GENERATOR
    ```
2.  **Install dependencies:**
    ```bash
    npm install
    ```
3.  **Set up environment variables:**
    *   Create a `.env` file in the root directory.
    *   Add your Google Gemini API key:
        ```env
        GOOGLE_API_KEY=YOUR_GEMINI_API_KEY
        ```
        *(Remember to keep your API key secret!)*
4.  **Run the development server:**
    ```bash
    npm run dev
    ```
    The application will be available at `http://localhost:3000`.
