## ✨ Features

- **7-Step Form** — Identity · About · Skills · Stats · Social · Extras · Style
- **Live Preview** — See your rendered README update as you type
- **Markdown Export** — One-click copy or download as `README.md`
- **50+ Skill Badges** — Languages, frameworks, databases & DevOps tools
- **GitHub Stats** — Stats card, streak, top languages, trophies, activity graph, snake animation
- **Social Links** — LinkedIn, Twitter/X, Instagram, YouTube, Dev.to, Medium, LeetCode, Kaggle, Buy Me a Coffee
- **Typing SVG Banner** — Animated cycling text via `readme-typing-svg`
- **3 Header Styles** — Simple · Waving Banner · Animated Venom
- **10 Stats Themes** — Radical, Tokyo Night, Dracula, Nord, Gruvbox, and more
- **4 Badge Styles** — Flat Square, For The Badge, Plastic, Flat
- **Profile Views Counter**, **Dev Quote**, **WakaTime Stats**, **Spotify Now Playing** toggles
- **Custom Cursor** — Smooth animated cursor that reacts to interactive elements
- **Fully responsive** dark UI with dot-grid background and glassmorphism nav

## 🎨 How It Works

1. **Fill in your details** across the 7 step tabs on the left panel
2. Click **Generate README** (or just type — live preview updates automatically)
3. Switch between **👁 Preview** and **📄 Markdown** tabs on the right
4. Hit **Copy** or **Download README.md**
5. Paste the content into your GitHub profile repo (`username/username`)
